# Data Organization for Metagenomics Projects

This workshop teaches best practices for organizing, documenting, and managing metagenomic sequencing projects. You'll learn how to structure your data, create effective metadata, and work with public sequencing databases.

## What You'll Learn

- Principles of tidy data and metadata organization
- Best practices for structuring spreadsheets
- Planning and managing NGS (Next-Generation Sequencing) projects
- Storing and backing up sequencing data
- Accessing data from NCBI's Sequence Read Archive (SRA)
- Organizing project directories for reproducible analysis

## Why Data Organization Matters

- **Reproducibility:** Well-organized projects are easier to reproduce and share
- **Collaboration:** Clear organization helps collaborators understand your work
- **Efficiency:** Good structure saves time when analyzing data
- **Publication:** Organized metadata is essential for submitting data to repositories
