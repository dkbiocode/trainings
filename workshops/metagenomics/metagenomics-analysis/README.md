# Metagenomics Analysis Workshop

This workshop teaches shotgun metagenomics analysis, from raw sequencing reads to taxonomic and functional insights. You'll learn to assess read quality, perform assembly, bin contigs into genomes, and assign taxonomy using data from the Cuatro Ciénegas basin.

## What You'll Learn

- Quality control of metagenomic sequencing data (FastQC)
- Read trimming and filtering (Trimmomatic)
- Metagenomic assembly (SPAdes)
- Genome binning from metagenomes (MaxBin2, CheckM)
- Taxonomic assignment (Kraken2)
- Diversity analysis and visualization (R/phyloseq)
